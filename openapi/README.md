# OpenAPI Spec Snapshot

This directory stores a versioned local snapshot of the YNAB OpenAPI spec for tooling and agent workflows.

- Canonical source URL: `https://api.ynab.com/papi/open_api_spec.yaml`
- Local snapshot: `openapi/ynab-v1.openapi.yaml`
- Sync command: `./scripts/sync-openapi-spec.sh`

Do not hand-edit `openapi/ynab-v1.openapi.yaml`. Always refresh it from the canonical source with the sync script.
